# 八字健康倾向分析指南

## 这份文件做什么

这部分只适合做传统术数视角下的身体状态倾向、作息提醒与阶段变化分析，不替代医生诊断，也不提供治疗建议。

## 分析重点

健康倾向分析时，重点不在“断病名”，而在：
- 哪类状态容易失衡
- 哪个阶段更容易消耗
- 作息、情绪、压力是否会放大问题
- 结构中偏枯、偏燥、偏寒、偏滞的倾向
- 身体感受与生活节奏是否长期不协调

## 常见观察点

### 1. 五行偏枯
看是否有某一类力量长期不足或过盛。

### 2. 日主承受状态
看整体是否容易过劳、受压、失衡。

### 3. 大运流年变化
有些问题更像阶段性波动，不宜直接绝对化。

### 4. 生活方式影响
看问题是否可能被熬夜、饮食混乱、情绪波动或长期压力放大。

## 输出建议

可以按这个顺序讲：
1. 身体状态总体倾向
2. 更容易失衡的方向
3. 哪类生活节奏更需要注意
4. 阶段变化提醒
5. 更适合的调整方向
6. 医疗边界说明

## 表达原则

- 不写具体病名判断
- 不替代医疗建议
- 多讲失衡倾向、作息提醒和阶段变化
