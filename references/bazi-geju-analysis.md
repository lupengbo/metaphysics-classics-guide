# 八字格局分析指南

## 这份文件做什么

用于帮助技能在分析格局时，不只是套一个格局名字，而是把结构成立条件、强点、破点和适合的理解方式讲清楚。

## 分析重点

格局分析重点不在背名字，而在：
- 结构是否成立
- 格是否清
- 有没有被破
- 用神是否跟得上
- 格局是核心主线，还是只能作为辅助理解

## 常见观察点

### 1. 结构成立度
先看这个格局到底成不成，是明显成立，还是只是有接近倾向。

### 2. 结构强点
看这个格局真正站得住的地方在哪里。

### 3. 易破位置
看哪些关系、冲克、失衡最容易把结构打散。

### 4. 适合理解方式
看这个命局更适合从格局切入，还是更适合从整体平衡和流通理解。

## 输出建议

可以按这个顺序讲：
1. 更接近什么结构
2. 结构强点在哪里
3. 容易被破的点在哪里
4. 适合怎么看，不适合怎么看
5. 用神是否能配得上这个结构

## 表达原则

- 不迷信格局名字本身
- 多讲结构条件和成立程度
- 多讲为什么这样看，而不是只报术语
