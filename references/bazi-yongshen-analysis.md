# 八字用神分析指南

## 这份文件做什么

用于帮助技能在分析用神时，不只是报出一个五行或一个字，而是把命局真正缺什么、为什么缺、该怎么理解讲清楚。

## 分析重点

用神分析不是只报一个字，而要讲清：
- 命局当前最需要什么
- 为什么需要它
- 是扶抑、调候还是通关思路
- 用神能不能真正立得住
- 阶段变化时，用神重点会不会发生偏移

## 常见观察点

### 1. 原局主要问题
先看命局最核心的问题是偏弱、偏强、失衡、寒燥，还是卡在流通不顺。

### 2. 用神方向
看最需要补的是哪种力量，以及它为什么是当前最关键的一环。

### 3. 忌神位置
看哪些力量会继续放大问题，而不是帮忙平衡。

### 4. 阶段变化
看在不同大运流年下，用神重点会不会有所转移。

## 输出建议

可以按这个顺序讲：
1. 原局主要问题
2. 最需要补的方向
3. 用神思路
4. 忌神提醒
5. 阶段变化时是否会改变重点
6. 怎么理解这种取用更稳妥

## 表达原则

- 不要只报结果，不讲原因
- 多讲为什么这样取，而不是机械背法
- 多讲用神与整体结构的关系
