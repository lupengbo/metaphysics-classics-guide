# 八字婚姻分析指南

## 这份文件做什么

用于帮助技能在分析婚姻问题时，不只是说“成不成”或“稳不稳”，而是把关系稳定度、相处节奏、现实压力和阶段变化讲清楚。

## 分析重点

婚姻分析重点不是一句“成不成”，而是看：
- 稳定度
- 相处节奏
- 现实压力
- 阶段变化
- 哪类问题更容易长期影响关系

## 常见观察点

### 1. 关系稳定度
看关系是偏稳、偏反复，还是需要经历较长磨合。

### 2. 相处节奏
看双方更适合慢慢推进，还是容易一快就失衡。

### 3. 现实压力
看婚姻问题是否更容易受现实事务、责任分配、生活节奏影响。

### 4. 阶段变化
看变化是阶段性波动，还是更深层的结构问题。

## 输出建议

可以按这个顺序讲：
1. 婚姻结构总体倾向
2. 关系稳定度
3. 容易受影响的因素
4. 更适合的相处方式
5. 阶段提醒
6. 更适合的沟通重点

## 表达原则

- 少做命定式判断
- 多讲关系经营和现实节奏
- 多提醒如何稳住关系，而不是只报结果
