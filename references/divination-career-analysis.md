# 占卜事业问题分析指南

## 这份文件做什么

用于帮助技能在处理事业类占卜问题时，不只是说“值不值得做”，而是把当前阶段、阻力与机会、时间窗口和行动建议讲清楚。

## 分析重点

事业类占卜重点看：
- 这件事值不值得推进
- 当前最大阻力是什么
- 后续有没有机会窗口
- 更适合进还是守
- 问题更偏内部判断，还是外部条件限制

## 常见观察点

### 1. 当前阶段判断
先看这件事现在是顺势推进、卡住不动，还是需要重新判断方向。

### 2. 阻力与机会
看问题是来自资源、节奏、对方、规则，还是自己判断。

### 3. 时间窗口
看现在是不是合适时机，还是更适合再等一轮。

### 4. 动作选择
看更适合继续推进、调整方式，还是先守住不乱动。

## 输出建议

可以按这个顺序讲：
1. 问题确认
2. 当前阶段判断
3. 阻力与机会
4. 时间窗口
5. 建议
6. 边界提醒

## 表达原则

- 不直接替用户拍板
- 多讲节奏、条件和窗口
- 多给方向性建议，不装作确定无误
