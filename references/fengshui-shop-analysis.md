# 店铺风水分析指南

## 这份文件做什么

用于帮助技能在分析店铺时，不只是说位置吉凶，而是把门头、人流、动线、展示区和留客能力讲清楚。

## 分析重点

店铺分析重点看：
- 门头是否清晰
- 人流是否容易进入
- 动线是否能留人
- 核心展示区是否突出
- 收银区与核心区关系是否顺
- 空间是否容易让人进来、停留、转化

## 常见观察点

### 1. 门头与入口
看门头是否明确、入口是否好找、是否容易吸引人进入。

### 2. 人流进入方式
看顾客是容易自然进入，还是会被入口、通道或布局挡住。

### 3. 动线与停留
看进店之后的路线是否顺，是否能自然引导顾客停留和浏览。

### 4. 核心区域关系
看主展示区、收银区、重点商品区之间的关系是否协调。

### 5. 空间问题优先级
看哪些问题只是细节，哪些问题会直接影响进店率与停留感。

## 输出建议

可以按这个顺序讲：
1. 已知条件
2. 主要观察
3. 有利点
4. 不利点
5. 可调整点
6. 优先处理建议

## 表达原则

- 多讲动线、停留和空间转化感
- 少讲空泛玄词
- 多给能落地的调整建议
