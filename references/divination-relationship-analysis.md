# 占卜感情问题分析指南

## 这份文件做什么

用于帮助技能在处理感情类占卜问题时，不只是说“对方怎么想”，而是把关系状态、阻力点、转机窗口和建议讲清楚。

## 分析重点

感情类占卜重点看：
- 对方态度
- 当前关系是推进还是停滞
- 矛盾主要卡在哪
- 后续有没有转机
- 问题更偏情绪性，还是现实性

## 常见观察点

### 1. 当前关系状态
先判断现在是靠近、拉扯、冷却，还是停滞阶段。

### 2. 对方态度
看对方是真犹豫、真退缩，还是只是阶段性观望。

### 3. 阻力点
看问题出在误会、现实压力、第三方因素，还是沟通失衡。

### 4. 转机窗口
看后续还有没有重新沟通、重新靠近或重新确认关系的机会。

## 输出建议

可以按这个顺序讲：
1. 问题确认
2. 当前关系状态
3. 对方态度倾向
4. 阻力点
5. 转机点
6. 建议
7. 边界提醒

## 表达原则

- 少神断式下结论
- 多讲关系状态和节奏
- 多讲阻力与转机，不只报结果
