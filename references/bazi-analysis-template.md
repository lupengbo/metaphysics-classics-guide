# 八字 / 四柱分析模板

## 适用触发词

- 算八字
- 看八字
- 批八字
- 排八字
- 四柱分析
- 命盘分析
- 算命
- 排盘

## 推荐输入模板

当用户要做八字 / 四柱分析时，优先引导其提供：

1. 出生日期（公历优先）
2. 出生时间（尽量精确到小时）
3. 性别
4. 出生地（城市即可，用于时区/历法换算时参考）
5. 想重点看的方向（整体、事业、财运、感情、学业、阶段走势）

可直接给用户这种输入模板：

```text
出生日期：1998-08-16
出生时间：14:30
性别：男
出生地：湖南长沙
想重点看：事业和近两年走势
```

如果用户信息不完整，要先说明哪些判断会受影响。

## 推荐输出结构

### 1. 基本信息
- 出生信息是否完整
- 当前采用什么分析框架（如子平命理 / 四柱八字）
- 信息不完整时，先提示局限

### 2. 四柱结构
- 年柱
- 月柱
- 日柱
- 时柱
- 先做结构展示，再进入分析

### 3. 核心判断
优先覆盖：
- 日主强弱
- 五行分布
- 偏旺偏弱的方向
- 结构是否均衡
- 明显特征点

### 4. 展开分析
按用户问题选择展开：
- 整体性格与倾向
- 事业/学业
- 财运思路
- 感情婚姻倾向
- 家庭关系倾向
- 阶段走势分析

### 5. 走势分析
如果用户问“最近”“明年”“未来几年”：
- 先给阶段判断
- 再说变化点
- 再说有把握与没把握的部分

### 6. 输出边界
结尾默认补一句：

> 以上内容为传统术数框架下的分析草稿，适合做文化研究、娱乐参考与个人思考，不构成现实中的医疗、法律、投资等专业建议。

## 推荐话术风格

建议采用：
- 先讲依据，再讲结论
- 少用绝对语气
- 把“高概率倾向”和“可能性判断”分开
- 不用恐吓式表达

## 经典引用示例

输出时，可以像下面这样把依据自然融进去：

- “按《渊海子平》的常见看法，四柱分析先重月令，再看日主强弱与十神配置。”
- “从《子平真诠》的格局思路看，这里重点不只是看旺衰，还要看格局有没有成，以及用神有没有被破。”
- “若按《滴天髓》强调的气势与流通来看，这类结构更怕五行阻滞，而不是单纯某一行多少。”
- “《三命通会》里常把类似组合放到更大的类例中去看，所以这里适合做补充比照，不宜孤立下断语。”

不要硬塞大段古文，重点是：
- 让用户知道判断不是凭空来的
- 让经典依据自然服务于分析
