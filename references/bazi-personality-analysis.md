# 八字性格倾向分析指南

## 这份文件做什么

用于帮助技能在分析性格倾向时，不只是贴标签，而是把表达方式、做事风格、压力反应和人际互动讲清楚。

## 分析重点

性格分析不要写成标签化人格测试，而应重点看：
- 表达方式
- 情绪节奏
- 对压力的反应
- 做事风格
- 与人互动方式
- 哪些地方更容易失衡

## 常见观察点

### 1. 表达方式
看更偏直接、克制、谨慎，还是容易外放。

### 2. 情绪节奏
看情绪是稳定、内压，还是容易起伏明显。

### 3. 压力反应
看遇到压力时更容易硬扛、逃避、内耗，还是主动调整。

### 4. 人际互动
看是更独立、依附、敏感，还是更擅长合作与协调。

## 输出建议

可以按这个顺序讲：
1. 性格总体倾向
2. 做事方式
3. 人际互动特点
4. 容易失衡的点
5. 适合的调整方向
6. 哪些场景更适合发挥优势

## 表达原则

- 不要写成人格标签定型
- 多讲倾向和模式
- 多讲优势与失衡点并存
