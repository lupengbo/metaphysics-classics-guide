# 经典分析依据层

这个文件不是机械照搬外部 skill，而是把传统术数分析时常用的“依据层”整理成你自己的版本，供技能在分析时优先参考。

## 适用原则

当用户要求：
- 算八字
- 看四柱
- 看命盘
- 做命理分析
- 做占卜解读
- 看走势预测

优先按以下顺序组织依据：

1. **基础框架**
   - 《周易》
   - 《五行大义》
   - 《皇极经世》
   - 《黄帝阴符经》

2. **命理核心依据**
   - 《渊海子平》
   - 《三命通会》
   - 《子平真诠》
   - 《滴天髓》

3. **占卜与走势依据**
   - 《梅花易数》
   - 《增删卜易》
   - 《周易》

4. **风水与形势补充**
   - 《葬书》
   - 《地理人子须知》
   - 《青囊奥语》
   - 《撼龙经》

## 分析时的默认做法

### 八字 / 四柱分析
优先用：
- 《渊海子平》做四柱命理基础框架
- 《子平真诠》做格局与用神整理
- 《滴天髓》做旺衰与气势理解
- 《三命通会》做补充条目与对照参考

### 占卜解读
优先用：
- 《周易》做象义母体
- 《梅花易数》做快速取象与外应思路
- 《增删卜易》做六爻与案例化判断思路

### 走势预测
优先用：
- 命理类：看结构 + 阶段变化 + 喜忌倾向
- 易学类：看变化节点、阻力、转机与趋势方向
- 《皇极经世》可作为“周期/阶段变化”补充视角，但不要装作精确算法

## 输出要求

回答时尽量体现：
- 主要依据来自哪些经典脉络
- 是“综合整理”还是“偏某一派”
- 哪部分判断把握更高，哪部分是推测性延伸

## 不建议的写法

避免这种句子：
- “你命里注定如此”
- “百分之百就是这样”
- “一定会发生”

改成：
- “按传统子平命理的常见理解”
- “从《滴天髓》强调的旺衰和气势角度看”
- “若按《梅花易数》的取象思路”
- “综合这几部典籍的常见解释，这类结构更偏向……”
