# 占卜第三方因素分析指南

## 这份文件做什么

用于帮助技能在分析占卜问题时，把第三方干扰因素讲清楚，不把所有问题都归结为双方本身。

## 分析重点

有些占卜问题并不是双方之间的问题，而是被第三方因素影响。

常见第三方因素包括：
- 其他人的意见
- 外部资源变化
- 家庭因素
- 规则限制
- 时机未到

## 常见观察点

### 1. 干扰来源
先看第三方影响更来自人、资源、规则，还是环境条件。

### 2. 干扰强度
看它只是短期插手，还是已经成为核心卡点。

### 3. 干扰性质
看是误会型干扰、现实型限制，还是长期结构性影响。

### 4. 后续缓和空间
看这种影响后面有没有松动、缓和或绕开的机会。

## 输出建议

可以按这个顺序讲：
1. 当前是否存在第三方干扰
2. 干扰来自哪里
3. 是短期影响还是结构性影响
4. 后续是否有缓和空间
5. 更适合的应对方式

## 表达原则

- 不把所有问题简单甩给第三方
- 多讲影响路径和缓和空间
- 多讲当前真正的核心卡点
