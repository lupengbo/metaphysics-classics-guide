# 风水分析模板

## 适用触发词

- 风水分析
- 看风水
- 家居风水
- 办公室风水
- 阳宅风水
- 阴宅风水
- 格局怎么看
- 这个位置风水怎么样

## 推荐输入模板

做风水分析时，优先让用户提供：

1. 分析对象（住宅 / 办公室 / 店铺 / 具体空间）
2. 大致位置或朝向
3. 当前格局描述
4. 想重点看的问题（财位、事业、居住舒适度、动线、采光、布局）
5. 如果有图片、平面图、户型图，也可一并提供

示例：

```text
分析对象：住宅
朝向：坐北朝南
当前情况：入户见客厅，厨房在西北角，卧室在东南角
想重点看：整体格局和财位
```

## 推荐输出结构

### 1. 对象确认
- 看的是什么空间
- 已知信息是否足够
- 哪些判断依赖更多现场信息

### 2. 所用框架
- 形势观察
- 方位与动线
- 空间关系
- 如果用户要求更传统，可说明参考《葬书》《地理人子须知》《青囊奥语》《撼龙经》的形势思路

### 3. 主要观察点
优先看：
- 入口与气口
- 动线是否顺
- 明暗与开阔感
- 功能区位置关系
- 明显冲突点

### 4. 分析结论
建议拆成：
- 整体格局倾向
- 有利点
- 不利点
- 可调整点

### 5. 优化建议
- 哪些地方可以先改
- 哪些判断需要更多图纸或现场信息

### 6. 边界说明

> 以上内容为传统风水与空间关系视角下的分析草稿，仅供文化研究、空间参考与个人思考使用，不构成建筑、法律、医疗等专业意见。

## 经典引用示例

- “按《葬书》强调的‘藏风聚气’思路，这里更重要的是空间是否能够聚而不散。”
- “从《地理人子须知》的形势观察角度看，这类格局重点在外部环境与内部关系是否协调。”
- “若借《青囊奥语》的传统表达，关键不在单个方位好坏，而在整体配合是否顺。”
- “《撼龙经》更强调形势和脉络，所以遇到风水类问题时，尽量不要只看一个点位下结论。”
