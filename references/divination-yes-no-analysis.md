# 占卜是否类问题分析指南

## 这份文件做什么

用于帮助技能在处理“能不能成”“是不是”“会不会”这类问题时，把结果倾向、成立条件和变化点讲清楚，避免说得太死。

## 分析重点

“能不能成”“是不是”“会不会”这类问题，最容易被说得太死。

更稳妥的做法是：
- 先给结果倾向
- 再讲条件
- 再讲变化点
- 再讲当前为什么还不能完全定死

## 常见观察点

### 1. 结果倾向
先判断整体更偏肯定、否定，还是仍在摇摆。

### 2. 成立条件
看结果成立需要哪些前提，哪些条件没到位就很难落地。

### 3. 阻力点
看问题是卡在时间、态度、资源，还是外部因素。

### 4. 转机点
看后续是否有重新推动、重新确认或条件改变的机会。

## 输出建议

可以按这个顺序讲：
1. 结果倾向
2. 成立条件
3. 阻力点
4. 转机点
5. 建议
6. 边界提醒

## 表达原则

- 不直接神断
- 多讲条件和变化空间
- 多讲为什么现在只能给倾向，不给绝对结论
