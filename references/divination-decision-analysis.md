# 占卜决策问题分析指南

## 这份文件做什么

用于帮助技能在处理“要不要做”“该不该继续”“要不要换方向”这类问题时，把利弊、风险和节奏讲清楚，而不是替用户武断拍板。

## 分析重点

当用户问“要不要做”“该不该继续”时，重点不是替用户做决定，而是帮助拆开：
- 当前利弊
- 主要阻力
- 风险点
- 更适合等、进还是换方向
- 做决定前还缺什么关键信息

## 常见观察点

### 1. 当前局面
先看这件事现在更像顺势推进、勉强硬推，还是已经明显失衡。

### 2. 利弊对比
看继续做的好处是什么，代价又是什么。

### 3. 风险位置
看问题更可能出在时间、条件、资源、对方态度，还是自己判断。

### 4. 决策节奏
看是现在就动比较合适，还是再等等、再观察一轮更稳。

## 输出建议

可以按这个顺序讲：
1. 当前局面
2. 有利点
3. 不利点
4. 更适合的动作
5. 还需要确认的点
6. 边界提醒

## 表达原则

- 不替用户下命令式结论
- 多讲利弊、节奏和条件
- 多给方向性建议，不装作全知
