# 占卜复合与和解分析指南

## 这份文件做什么

用于帮助技能在分析复合、和解、重新联系这类问题时，把关系状态、阻力性质和窗口期讲清楚，而不是一句“能不能复合”就结束。

## 分析重点

这类问题重点不是一句“能不能复合”，而是看：
- 双方还有没有连接
- 当前阻力是情绪性的还是结构性的
- 后续有没有重新沟通窗口
- 问题是阶段性冷却，还是已经走到很难回头的状态

## 常见观察点

### 1. 当前关系状态
先看双方是完全断开、表面冷却，还是还有潜在联系。

### 2. 阻力性质
看主要问题是情绪未消、误会未解，还是现实条件根本没变。

### 3. 后续窗口
看是不是还有重新接触、重新表达、重新协调的机会。

### 4. 关系修复条件
看如果想复合或和解，需要先解决什么，而不是只问结果。

## 输出建议

可以按这个顺序讲：
1. 当前关系状态
2. 复合或和解可能性倾向
3. 当前主要阻力
4. 后续窗口
5. 建议
6. 边界提醒

## 表达原则

- 少做绝对式判死刑
- 多讲关系状态和修复条件
- 多讲窗口期，而不是只讲能不能
