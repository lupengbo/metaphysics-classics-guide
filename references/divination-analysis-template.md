# 占卜分析模板

## 适用触发词

- 占卜
- 起卦
- 看卦
- 梅花易数
- 六爻
- 卜一卦
- 问一下这个事情怎么发展
- 这个事能不能成

## 推荐输入模板

做占卜解读时，优先让用户提供：

1. 问题本身（越具体越好）
2. 起卦时间 / 提问时间
3. 当前背景信息
4. 想重点看的维度（能不能成、何时变化、阻力在哪、对方态度、后续走势）
5. 如果用户已有卦象/卦名/爻位，也一起给出

示例：

```text
问题：这个合作项目还能不能继续推进？
时间：2026-04-07 17:00
背景：已经谈了两轮，对方最近回复变慢
想重点看：还能不能成，卡点在哪
```

## 推荐输出结构

### 1. 问题确认
- 用户问的是什么
- 问题范围是否明确
- 信息是否足够

### 2. 所用方法
- 梅花易数 / 六爻思路 / 易象取象 / 综合传统框架
- 如果没有完整起卦条件，要说明这是“分析草稿”而非严格技术排盘

### 3. 占断依据
优先解释：
- 主要信号
- 阻力信号
- 转机信号
- 时间/节奏信号

### 4. 结论输出
建议拆成：
- 结果倾向
- 当前状态
- 后续变化点
- 关键影响因素

### 5. 行动建议
- 如果偏成，建议怎么顺势推进
- 如果阻力大，建议观察什么信号
- 如果结论不明，说明为什么不明

### 6. 边界说明

> 以上内容为传统占卜/易学框架下的解读草稿，仅供研究、娱乐与个人参考，不构成现实中的法律、投资、医疗等专业建议。

## 推荐话术风格

- 先确认问题，再做判断
- 少用吓人的词
- 不把“短期阻力”直接说成“彻底失败”
- 把“倾向”“节点”“变数”分开讲

## 简版骨架

```text
一、问题确认
- 你这次问的是……

二、占断依据
- 从传统易学视角看，当前最强的信号在……
- 阻力主要体现为……
- 变化点可能出现在……

三、结论
- 结果倾向：……
- 当前状态：……
- 后续走势：……

四、建议
- 现阶段更适合……

五、边界说明
- 以上为传统术数解读，仅供参考。
```
