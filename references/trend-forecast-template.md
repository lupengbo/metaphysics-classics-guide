# 走势预测模板

## 适用触发词

- 走势预测
- 趋势分析
- 后面会怎么样
- 接下来怎么走
- 未来几个月走势
- 明年运势
- 未来发展趋势

## 使用范围

这个模板适合：
- 感情走势
- 事业走势
- 财运走势
- 学业走势
- 某件事的阶段变化

不适合：
- 保证收益
- 绝对结果承诺
- 医疗诊断替代
- 法律结果保证

## 推荐输入模板

1. 分析对象（人 / 关系 / 项目 / 事件）
2. 当前状态
3. 想看的时间范围（近期 / 半年 / 一年 / 某个具体阶段）
4. 想看的方向（事业 / 财运 / 感情 / 综合）
5. 如果有命盘、卦象、四柱等，也可一并提供

示例：

```text
分析对象：我的工作发展
当前状态：最近在考虑换岗
时间范围：未来一年
想重点看：事业走势和变化点
```

## 推荐输出结构

### 1. 分析对象确认
- 看的是谁 / 什么事
- 时间范围是什么
- 分析边界是什么

### 2. 所用视角
- 传统命理视角
- 占卜/易学视角
- 综合术数视角
- 如果信息不足，要先说明是“趋势草稿”而非严格推盘结果

### 3. 阶段划分
走势分析尽量拆成阶段：
- 当前阶段
- 过渡阶段
- 后续阶段

### 4. 每阶段重点
每一阶段尽量给：
- 主基调
- 有利因素
- 阻力因素
- 变化点

### 5. 总结判断
- 整体偏上行 / 震荡 / 回落 / 反复
- 哪一段最关键
- 结果受什么因素影响最大

### 6. 边界说明

> 以上内容为传统术数视角下的走势分析草稿，仅供研究、娱乐与个人参考，不构成医疗、法律、投资等专业建议。

## 推荐话术风格

- 用“阶段变化”代替“绝对结论”
- 用“倾向”代替“保证”
- 说明哪些判断更有把握，哪些部分只是推测

## 经典引用示例

走势分析时可以这样带依据：

- “按《滴天髓》强调的旺衰与流通思路，这类走势判断不能只看一时高低，还要看整体气势是否顺。”
- “从《三命通会》的类例补充角度看，这类结构往往不是直线上升，而是先有波动再出现转机。”
- “若借《皇极经世》的阶段变化视角，这类问题更适合拆成几个阶段来看，而不是一句话定终局。”

## 简版骨架

```text
一、对象与范围
- 这次看的重点是……
- 时间范围是……

二、阶段走势
- 当前阶段：……
- 过渡阶段：……
- 后续阶段：……

三、关键变化点
- 按《滴天髓》重气势的角度看，真正的转折点更可能在……

四、总结
- 整体走势偏……
- 需要重点注意……

五、边界说明
- 以上为传统术数视角下的分析，仅供参考。
```
