# 占卜规则摘要

## 1. 问题先收敛

占卜类请求不要一上来就断。
先确认：
- 问题具体是什么
- 问的是结果、过程还是对方态度
- 时间范围是什么

问题越具体，解读越容易落地。

## 2. 判断重点

占卜分析通常更适合看：
- 结果倾向
- 当前阻力
- 转机点
- 时间节奏

不适合包装成绝对确定的现实承诺。

## 3. 取象思路

做解读时，可以结合：
- 问题本身
- 提问背景
- 时间信息
- 象义与变化点
- 外应（若用户明确提供）

## 4. 输出原则

建议拆成：
1. 问题确认
2. 占断依据
3. 结果倾向
4. 变化点
5. 建议
6. 边界说明

## 5. 常见错误

- 问题太大太空
- 一句神断，不解释为什么
- 把短期阻力说成彻底失败
- 把倾向判断说成命定结局
