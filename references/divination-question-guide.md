# 占卜问法指南

## 为什么要先整理问题

占卜类请求最怕的问题不是“不会断”，而是用户问得太散、太大、太空。

所以技能要先帮助用户把问题收敛。

## 更适合问的问题

### 结果类
- 这个合作还能不能成？
- 这次面试通过概率大吗？
- 这件事继续推进值不值得？

### 过程类
- 目前最大的阻力在哪？
- 这个阶段最该注意什么？
- 这件事后面会怎么变化？

### 时间类
- 未来一个月会不会有转机？
- 今年下半年走势怎么样？

## 不适合直接问的问题

- 我这一辈子会怎么样？
- 我命里到底是不是这样？
- 我是不是注定失败？
- 能不能百分之百发财？

这类问题太大、太死，技能应引导用户缩小范围。

## 推荐引导方式

### 如果问题太大
可以回：

```text
这个问题太大了，我建议你先缩小到一个具体方向。
比如你是想看：
1. 这件事能不能成
2. 当前阻力在哪
3. 后面一段时间怎么发展
```

### 如果问题太空
可以回：

```text
可以看，但你先告诉我：
- 你具体问的是什么事
- 目前是什么状态
- 你最想看结果、过程，还是时间点
```

## 技能内部原则

1. 先收问题
2. 再收背景
3. 再收重点方向
4. 最后才分析

## 结论表达建议

- 用“倾向”代替“绝对”
- 用“阶段变化”代替“一锤定音”
- 用“可能的阻力/转机”代替“神断式结局”
