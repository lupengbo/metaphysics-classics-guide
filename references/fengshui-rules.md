# 风水规则摘要

## 1. 先整体，后局部

做风水分析时，优先看：
- 整体空间关系
- 入口与气口
- 动线是否顺
- 明暗与开阔度
- 功能区之间是否冲突

不要只盯一个角落下结论。

## 2. 形势优先

传统风水里，很多时候先看：
- 外部环境
- 内部格局
- 空间聚散
- 人在空间里的使用关系

## 3. 常见可分析点

适合在远程草稿里分析：
- 户型大致格局
- 门、窗、通道关系
- 厨房、卧室、办公位的大致位置
- 动线和采光感受

## 4. 常见不可过度判断点

如果没有现场信息，不要装作能精确判断：
- 极细的点位吉凶
- 精确到分度的朝向
- 外部复杂环境的全貌

## 5. 输出原则

建议拆成：
1. 已知条件
2. 主要观察
3. 有利点
4. 不利点
5. 可调整点
6. 边界说明

## 6. 风格要求

- 少堆玄词
- 多讲空间关系
- 多讲可调整点
- 明确哪些判断需要更多资料
